# BI-Portfolio
Portfólio em construção com projetos voltados à estruturação de dados em SQL e aplicação de Business Intelligence, com direcionamento para análise de performance e planejamento financeiro (FP&A).

Cada pasta representa um projeto estruturado a partir de um dataset que permita análises financeiras e gerenciais, contendo, preferencialmente:
- Receita
- Custos
- Despesas
- Margem
- Período temporal consistente
- Categoria ou centro de custo

---

## Estrutura Metodológica dos Projetos
### 1. Contexto de Negócio
Cada projeto inicia com a definição do contexto empresarial, contendo:
- Apresentação do negócio
- Objetivos da análise
- Perguntas-chave que direcionam a investigação
O dataset é inserido neste contexto, servindo como base para responder às questões estratégicas propostas.

---

### 2. Modelagem e Estruturação dos Dados (SQL)
A etapa seguinte envolve a preparação e modelagem dos dados no SQL Server, incluindo:
- Criação das tabelas
- Inserção e organização dos dados
- Limpeza e padronização
- Modelagem dimensional (Star Schema)
- Criação de tabelas fato e dimensões
- Definição de chaves estrangeiras
- Criação de views analíticas
- Implementação de índices
- Queries exploratórias e analíticas
O objetivo desta fase é garantir consistência, integridade e base estruturada para análise.

---

### 3. Visualização e Definição de KPIs (Power BI)
Com os dados estruturados, desenvolve-se o dashboard analítico no Power BI, contemplando:
- Definição conceitual dos KPIs
- Implementação técnica das métricas
- Visualização estratégica dos dados
- Storytelling orientado ao negócio
- Análise de performance

---

### 4. Análise Gerencial e Financeira
A etapa final consolida os insights obtidos, incluindo:
- Análise de margem (bruta e operacional)
- Evolução de despesas
- Comparativos temporais
- Rolling forecast simplificado
- Interpretação gerencial e implicações estratégicas
