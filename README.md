# BI-Portfolio
Portfólio em construção com projetos voltados à estruturação de dados em SQL e aplicação de Business Intelligence, com direcionamento para análise de performance e planejamento financeiro.

Cada pasta representa um projeto estruturado a partir de um dataset que permita análises financeiras e gerenciais.

---

## Estrutura dos Projetos
Cada pasta do repositório representa um projeto de análise exploratória.

Os projetos possuem uma estrutura simples, contendo:
- README.md – Contexto do dataset, objetivo da análise e perguntas orientadoras
- Modelagem e Estruturação dos Dados (SQL) - Envolve a preparação e modelagem dos dados no SQL Server
- Visualização e Definição de KPIs (Power BI) - Com os dados estruturados, desenvolve-se o dashboard analítico no Power BI
- Insights.md – Principais interpretações e conclusões obtidas a partir da análise

---

## Objetivo do Repositório

Construir projetos que integrem dados, modelagem e visualização, aplicando Business Intelligence como suporte à análise de performance e à tomada de decisão. Os projetos deste repositório buscam demonstrar a aplicação prática de SQL e ferramentas de BI na estruturação de dados, definição de métricas e desenvolvimento de dashboards, com foco em traduzir informações em uma visão clara e objetiva do que está acontecendo no negócio.
