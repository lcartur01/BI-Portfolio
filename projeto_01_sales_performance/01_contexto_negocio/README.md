# Contexto do negócio

## Visão Geral

Um e-commerce norte-americano que comercializa diversos produtos em todo o território dos Estados Unidos decidiu realizar uma análise de dados utilizando informações de vendas referentes ao início de 2026.

O dataset disponível apresenta algumas inconsistências e problemas de qualidade de dados. Dessa forma, foi necessário realizar um processo inicial de limpeza, estruturação e modelagem das informações antes da etapa de análise.

A partir desse processo, foi conduzida uma análise com foco em indicadores financeiros e no desempenho geral das vendas.

---

## Objetivo da análise

Observar o desempenho geral do negócio.

---

## Fonte do dataset

O dataset pode ser acessado no Kaggle, com nome de 'E-commerce Sales Dataset' no link: https://www.kaggle.com/datasets/sharmajicoder/e-commerce-sales-dataset
